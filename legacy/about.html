About

Hi, I am Oliver Caldwell. I am a frontend web developer from England. I build great sites using HTML, CSS, JavaScript, PHP and Python. With a little help from Kohana, WordPress and the wonderful Django.

You can see more about my career and expertise on my StackOverflow careers page. It is a great collation of my work.

This blog is a place for me to help others and myself by writing about techniques and methods within web development. My posts will usually contain information about the following topics.

  • HTML
  • CSS
  • LESS
  • JavaScript
  • MooTools
  • WordPress
  • GitHub
  • Optimisation and validation
  • Python
  • Django

I try to make all of my JavaScript validate in JSHint. So I should be setting a good example for anyone reading. I will attempt to update at least once a week, usually twice, so please follow via RSS if you enjoy my posts.

If you want to know a little bit more about me, I am sure my Twitter, GitHub and StackOverflow accounts should suffice. If not, why not get in touch.

If you look at my GitHub profile, you will see that I love writing open source code, hence why this blog is open source.
